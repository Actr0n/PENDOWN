Introduction

While Markdown's redactional simplicity could be at the root of its success, unfortunately it also brings several quirks and limitations which may somewhat impede its usability in some cases.

This is probably why it has so many "flavors" trying to amend it in different ways.

While remaining in the same vein as its spiritual ancesort, Pendown is actually quite different in its purpose.

Like Markdown, it's a lightweight markup language, but its syntax was also designed with presentational features in mind, like text colors, font sizes, etc.

This means that generally it's absolutely not needed to use standard HTML tags along with these short tags, as all these features were already integrated in the language.

Syntax

Although Pendown was initially inspired from Markdown, most of its tags are actually quite different, as they were chosen according to different goals.

Styling

The presentation of the document content (texts, images, etc) can easily be tweaked by using any available CSS class, including those predefined in pendown.css.

This can be done by adding custom lists of classes (or properties) right after the Pendown tags :

{{^big,red,yellow_background\ a big red text with a yellow background }}

{{^@2.5,$00f,#eee,black_outline\ a big blue text with a light gray background and a black outline }}
You can also give a name to those lists, if you want to reuse them further in the document : 
{{^blue,italic:gangnam\ a blue text in italics }}

!^gangnam\ This title also uses the gangnam style.

Usage

Pendown tags can be converted to HTML directly by the web browser when the file is opened, which is especially convenient during its writing.

Here is a example of a HTML file whose content will be written using Pendown tags :

<meta charset="utf8"/>
<link rel="stylesheet" href="pendown.css">
<xmp>
! This is the title
    
Put your **Pendown** text here :)
</xmp>
<script src="pendown.js"></script>

Pendown documents can also be converted offline, using a command line tool :

For instance, the following command will convert all the Pendown tags of input.pd into standard HTML tags inside output.html. 

pendown --process --style input.pd output.html

Conclusion

Although Pendown isn't really a Markdown variant, it is unarguably a powerful alternative, especially for those who want to also control the presentation of their documents as easily as they would do in a word processor.
